The application has a CI that build the app and deployed to AWS (S3 + cloudfront)

## Getting started

Start with creating `.env` file, copy the .env.sample and rename it to .env and change the values inside.

### Install packages

`npm install`

### Run the application

`npm run start`

### Run tests

`npm run test`

### Prod build

`npm run build`
